PART A (Answers)
1. What is your current username?

The username displayed by the whoami command. Example: student, ubuntu, or your login account.

2. What is UID?

UID (User ID) is a unique numerical identifier assigned to each Linux user.

3. What is GID?

GID (Group ID) is a unique numerical identifier assigned to a user group.

4. What information does /etc/passwd contain?

The /etc/passwd file stores user account information such as username, UID, GID, home directory, login shell, and account details.
