# Linux Security Research

## 1. Why are file permissions important?

File permissions control who can read, modify, or execute files. They help protect sensitive data, prevent unauthorized changes, and maintain system security.

## 2. What happens if sensitive files are given 777 permissions?

When a file has 777 permissions, every user can read, modify, and execute it. This can lead to data theft, accidental deletion, malware execution, and unauthorized changes.

## 3. What is the Principle of Least Privilege?

The Principle of Least Privilege means users should receive only the permissions necessary to perform their tasks. This minimizes security risks and reduces the impact of mistakes or attacks.

## 4. Why do organizations restrict user access?

Organizations restrict access to protect confidential information, reduce insider threats, comply with security policies, and prevent accidental or malicious damage to systems and data.
